Asteroids with Quadtrees

C++
Collision Detection

This is a recreation of the Asteroids arcade game I made in the olc::PixelGameEngine. I did this project because I wanted to become more familiar with how collision detection works between simple objects. This game also handles all of the collision checking through Quadtrees. When more than 'x' amount of objects get put into that quadtree, the quadtree subdivides into 4 equally divided sub-quadtrees. This happens recursively until all objects are placed into a quadtree.


Now that all the objects are inside a bounding box, I only have to do collision checks between objects in that bounding box, and not with any of the other objects in the scene.

Hardcore Pizza Delivery (Grapple Game)

C++
Unreal Engine
Team

In Hardcore Pizza Delivery, the player has to deliver pizza while swinging around various environments. This project is made with a mix of Unreal Engine 4’s blueprint system and C++. I was a part of a team consisting of nine juniors and seniors. On this project, I worked as a general programmer and working on features such as the checkpoints/out of bounds system, grapple collision checking through raycasting, and I made a visual indicator on the gun to see how much rope the player had left. I was able to do this using Unreal's material system. I also helped the artists with any problems they came across like blueprints and materials.

Canvas Drawing App

JavaScript
Canvas

Here is a web drawing application made using Javascript and the HTML canvas element. In this application you can use multiple tools, like a line tool, circle, and square tool, to create a drawing. The application also supports layering. Each layer can be toggled on and off anytime as well as add and deleting layers while keeping the other layer's integrety. If the user doesn't like what they've drawn or made a mistake, there is also a undo and clear canvas button. These buttons undo the last action or clear all drawings on the current layer the user has selected. You can try out the application in a new window here.
