Asteroids with Quadtrees

C++
olc::PGE

This is a recreation of the Asteroids arcade game I made in the olc::PixelGameEngine. I did this project because I wanted to become more familiar with how collision detection works between simple objects. This game also handles all of the collision checking through Quadtrees. When more than 'x' amount of objects get put into that quadtree, the quadtree subdivides into 4 equally divided sub-quadtrees. This happens recursively until all objects are placed into a quadtree.


Now that all the objects are inside a bounding box, I only have to do collision checks between objects in that bounding box, and not with any of the other objects in the scene.
